Installation
============

The relevant packages can be installed from the Github repository found `here <https://github.com/phd-code/phd-code.github.io>`_. The 'phd' folder contains the code modules, the 'docs_source' folder contains the reStructuredText files used to build the documentation, and the 'docs' folder contains the built documentation. You will also need to install the `Computational Geometry Algorithms Library (CGAL) <https://www.cgal.org/>`_ (and its associated dependencies) following the instructions from their webpage. Once this Github repository is cloned locally, you can navigate to the home directory for the repo and use the following to install the code::

	pip install -e .
