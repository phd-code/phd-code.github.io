phd.utils.exchange_particles module¶

phd.utils.exchange_particles.exchange_particles(particles, send_data, send_particles, recv_particles, disp, comm, fields=None, offset_se=None, offset_re=None)¶

Exchange particles between processes. Note you have allocate the appropriate space for incoming particles in the particle container.

Parameters

  • particles (object) – particle container

  • send_data (ParticleArray) – particle carrays to send

  • send_particles (ndarray) – number particles to send to each process

  • recv_particles (ndarray) – number particles to receive from each process

  • disp (int) – starting index position for incoming particles

  • comm (object) – mpi controller

  • fields (list) – List of fields to export

phd.utils.run_parallel_script.run(filename, args=[], nprocs=2, timeout=20.0, path=None)¶

run a file python script

filename - filename of python script to run under mpi nprocs - (2) number of processes of the script to run (0 => serial non-mpi run) timeout - (5) time in seconds to wait for the script to finish running,

else raise RuntimeError exception

path - the path under which the script is located

Defaults to the location of this file (__file__), not curdir
