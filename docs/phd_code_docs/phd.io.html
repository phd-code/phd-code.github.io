read(file_name)¶

Read data and return particles.

For formats where the data is being stored, a read function must be defined. This function reads the data outputted by write and returns a CarrayContainer of particles to be used by the simulation.

Parameters

file_name (str) – File name to be read in.

Returns

particles – Complete container of particles for a simulation.

Return type

CarrayContainer

write(base_name, output_directory, integrator)¶

Write simulation to disk.

This base class is the api for writing and reading of data. Note the user is free to write any type of output. For example, this can be used to create runtime plots or diagnostics.

Parameters

  • base_name (str) – File name for output data.

  • output_directory (str) – Directory where to store the data.

  • integrator (IntegrateBase) – Advances the fluid equations by one step.
